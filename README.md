# React Value Change Playground

A React practice project focused on mastering **value-changing logic** using `useState`.

## 🚀 Features

- Counter
- Quantity Selector
- Rating System
- Progress Bar
- Font Size Changer
- Zoom In / Out
- Brightness Controller
- Volume Controller
- Opacity Controller
- Image Size Controller

## 📚 Concepts Practiced

- React Components
- `useState`
- Event Handling
- State Updates
- Conditional Rendering
- Dynamic Styling
- Inline Styles in React

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
